### 1. Для пользователя (Frontend):

1.1. **Поиск:**

-
    - На главной странице — одно поисковое поле.
    - Пользователь вводит название товара (например, "iPhone 15") и нажимает "Найти".

1.2. **Результаты поиска**

-
    - Страница со списком найденных товаров.
    - Для каждого товара в результатах отображается: 
    - - Миниатюра изображения.
    - - Название товара.
    - - Цена.
    - - Название магазина.
    - - Кнопка "Перейти в магазин" (ведет на оригинальную страницу товара).

1.3. **Страница товара**

-
    - При клике на товар из результатов поиска открывается детальная страница.
    - На ней отображается:
    - - Галерея изображений товара (1-2 картинки).
    - - Подробное описание.
    - - Таблица/список сравнения цен: Все предложения от разных магазинов на этот конкретный товар, отсортированные по цене (от низкой к высокой).
    - - Кнопки "Купить" или "Перейти в магазин" для каждого предложения.

### 2. Для администратора (Backend / Admin Panel):

2.1. **Управление товарами:**

- 
    - CRUD (Create, Read, Update, Delete) для товаров: добавление, редактирование, удаление.
    - Поля для товара: Название, Описание, Категория, Изображения.

2.2. **Управление магазинами:**

-
    - CRUD для магазинов-партнеров.
    - Поля для магазина: Название, Логотип, Домашний URL.

2.3. **Управление ценами:**

-
    - Возможность вручную или через простой скрипт (на этом этапе) добавить/обновить цену для связки "Товар-Магазин".
    - Поля: Товар (выбор из списка), Магазин (выбор из списка), Цена, URL товара в этом магазине.
