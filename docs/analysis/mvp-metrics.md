### 1. Парсинг цен (упрощенный):

-
    - На MVP можно временно обойтись ручным вводом данных через админку, чтобы сосредоточиться на основном функционале.
    - Следующим шагом будет написание простых парсеров (скриптов на Python с библиотеками requests и BeautifulSoup), которые будут собирать данные с сайтов-магазинов. Внимание: всегда проверяйте robots.txt и правила использования сайта.


### 2. Для администратора (Backend / Admin Panel):

-
    - Пользователь может найти товар.
    - Пользователь видит список предложений от разных магазинов.
    - Пользователь может перейти на сайт магазина по самой выгодной цене.
    - Администратор может добавлять новые товары и обновлять цены.
